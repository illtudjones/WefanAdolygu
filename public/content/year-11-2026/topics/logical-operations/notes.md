# Logical Operations

These revision notes are based on the WJEC GCSE Computer Science legacy specification (teaching from 2017 / award from 2019).

## Logical operators
- AND, OR, NOT and XOR operators are used in truth tables to solve logic problems.
- Learners should be able to combine logical operators and interpret the results.

## Boolean logic
- Boolean expressions can be simplified using Boolean identities and rules.
- Simplification reduces the number of logic gates or terms needed.
