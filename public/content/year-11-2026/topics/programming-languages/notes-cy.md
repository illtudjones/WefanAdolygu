# Ieithoedd rhaglennu

## Ieithoedd tagio
- Dylai dysgwyr ddylunio, ysgrifennu, profi a mireinio tudalennau HTML.
- Mae'r fanyleb yn enwi tagiau fel html, head, title, body, penawdau, paragraff, italig, trwm, alinio i'r canol, angori, mailto, rhestr heb drefn, eitem rhestr, dyfyniad bloc, llinell lorweddol a delwedd.

## Ieithoedd gwrthrych-gyfeiriadol
- Dylai dysgwyr ddylunio, ysgrifennu, profi a mireinio rhaglenni Java yn Greenfoot.
- Mae hyn yn cynnwys dosbarthiadau, gwrthrychau, bydoedd, dulliau, priodweddau, actorion, mewnbwn bysellfwrdd, seiniau, pasio paramedrau, canfod gwrthdrawiad, haprifau, etifeddu a mewngapsiwleiddio.

## Iaith gydosod
- Dylai rhaglenni cydosod syml ddefnyddio cofrifau fel INP, OUT, STA, LDA, ADD, SUB, BRA, HLT a DAT.
