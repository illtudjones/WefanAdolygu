# Communication

These revision notes are based on the WJEC GCSE Computer Science legacy specification (teaching from 2017 / award from 2019).

## Networks
- Different network types include LAN and WAN.
- Common topologies include ring, star, bus and mesh, each with advantages and disadvantages.
- Connectivity can be wired or wireless.
- Network communication may use circuit switching or packet switching.
- Important protocols include Ethernet, Wi‑Fi, TCP/IP, HTTP, HTTPS, FTP and email protocols.
- A TCP/IP packet contains items such as source and destination addresses, data and checks.
- The TCP/IP 5-layer model helps organise how communication works.
- Traffic can be routed across a network, and routing costs can be calculated.

## Internet
- DNS servers translate domain names into IP addresses.
- IP addresses identify devices on a network.
