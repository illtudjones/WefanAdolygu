# Hardware

## Architecture
- CPU architecture includes Von Neumann architecture, where data and instructions are stored in the same main memory.
- Key CPU components in the fetch-decode-execute cycle include the Control Unit, ALU, Program Counter, MAR, MDR, CIR and accumulator.
- CPU performance is affected by cache size, clock speed and number of cores.
- RISC processors use a small, simple instruction set; CISC processors use a larger, more complex instruction set.

## Input and output
- Input devices send data to the computer, such as keyboards, mice, microphones and sensors.
- Output devices present processed information, such as monitors, speakers, printers and actuators.

## Primary storage
- RAM is volatile working memory used for current programs and data.
- ROM is non-volatile and stores firmware or boot instructions.
- Flash memory is non-volatile and rewritable.
- Cache memory is very fast, small memory close to the CPU.

## Secondary storage
- Contemporary secondary storage includes magnetic, optical and solid-state storage.
- When comparing storage devices, consider suitability, durability, portability and speed.

## Storage requirements
- Storage units include bit, nybble, byte, kilobyte and larger prefix multipliers.
- Learners should be able to calculate data capacity requirements.

## Other hardware
- Other important hardware includes GPUs, sound cards and motherboards.

## Embedded systems
- Embedded systems are dedicated computer systems built into larger devices, such as washing machines, cars or traffic lights.
