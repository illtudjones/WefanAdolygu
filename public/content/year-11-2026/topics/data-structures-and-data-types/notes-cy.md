# Strwythurau data a mathau data

## Gweithredu strwythurau data
- Dylai dysgwyr ddefnyddio araeau un dimensiwn a dau ddimensiwn, ffeiliau a chofnodion.

## Gweithredu mathau data
- Dylai dysgwyr ddefnyddio mathau data cyfanrif, Boole, real, nod a llinyn.

## Newidynnau a chysonion
- Mae cysonion yn storio gwerthoedd nad ydynt yn newid.
- Mae newidynnau yn storio gwerthoedd a all newid wrth i raglen redeg.
- Dylid deall cwmpas a hyd oes newidynnau.
