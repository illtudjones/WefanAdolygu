# Meddalwedd system

Mae'r nodiadau adolygu hyn yn seiliedig ar hen fanyleb TGAU Cyfrifiadureg CBAC (addysgu o 2017 / dyfarnu o 2019).

## Rheoli adnoddau
- Mae'r system weithredu yn rheoli perifferolion, prosesau, cof a storfa gynorthwyol.

## Darparu rhyngwyneb
- Mae'r system weithredu yn darparu rhyngwyneb defnyddiwr fel bod pobl yn gallu rhyngweithio â'r cyfrifiadur.

## Meddalwedd gwasanaethu
- Mae meddalwedd gwasanaethu yn cyflawni tasgau cynnal a chadw fel gwneud copïau wrth gefn, gwrthfirysau, cywasgu ac offer disg.
