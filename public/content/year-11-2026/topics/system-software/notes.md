# System Software

These revision notes are based on the WJEC GCSE Computer Science legacy specification (teaching from 2017 / award from 2019).

## Managing resources
- The operating system manages peripherals, processes, memory and backing store.

## Providing an interface
- The operating system provides a user interface so people can interact with the computer.

## Utility software
- Utility software performs maintenance tasks such as backup, antivirus, compression and disk tools.
