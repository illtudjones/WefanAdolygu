# Digital Technology Systems

## Digital devices and interaction

- Users may interact with digital devices through speech, keyboard and mouse, gesture, touch, virtual reality, augmented reality and biometrics.

## Internet and web infrastructure

- The Internet is a global communications network using interconnected computers.
- Its infrastructure includes clients, servers and routers.
- Learners should understand the role of an ISP, search engines, the World Wide Web, web servers, URLs and how browsers request and display websites.

## Connection methods

- Internet-enabled devices may connect using broadband, satellite, fibre, mobile communication 4G/5G, Wi-Fi 802.11, Bluetooth and GIS.
- Learners should understand advantages and disadvantages of each method.

## Operating systems and HCI

- An operating system manages resources, peripherals, spooling, memory, processes, security and the user interface.
- Human computer interfaces include CLI, GUI, TSI, menu driven interfaces, biometrics and voice driven interfaces.

## Software, backup, cloud and SDLC

- Software types include applications software, bespoke software and process control software.
- Utility applications include task scheduling and antivirus software.
- Backup processes include full, incremental and differential backup, plus Grandfather-Father-Son methodology.
- Cloud services include cloud computing and cloud storage.
- The systems development life cycle has six parts: investigation, analysis, design, implementation, maintenance and evaluation.
